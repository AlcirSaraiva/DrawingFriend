# Drawing Friend

A tool for tracing reference images onto physical media using your mobile device's camera and screen.

## Core Functionality

**Drawing Friend** enables precise tracing by overlaying a semi-transparent reference image on your device's camera feed.

When positioned above paper or canvas, the reference image appears fixed in space relative to your physical surface, allowing accurate tracing.

---

## Key Technical Components

### Reference Image Overlay

* Load any image as a guide layer

### Transform Controls

* **Position** — Drag to place the reference image
* **Scale** — Pinch to zoom in/out
* **Rotation** — Two-finger rotate for alignment

### Opacity Adjustment

* 0–100% transparency slider for optimal visibility

### Camera Integration

* Real-time camera feed as the base layer

---

## Usage Workflow

1. Import a reference image
2. Adjust position, scale, and rotation to match the physical canvas
3. Set transparency to see both the reference and the canvas
4. Place the device in a holder above the paper
5. Trace while viewing the canvas through the screen
